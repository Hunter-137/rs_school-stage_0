How to play?

  • Use the arrow keys on your keyboard to move the tiles: ⇐ ⇑ ⇒ ⇓
  • To win the game, score 2048 points when merging tiles
  • If suddenly merging tiles becomes impossible, then click on the restart button and the game will start again
  • If you successfully fulfill the victory condition, a small window with congratulations will appear
